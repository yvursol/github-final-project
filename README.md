This project contains a Bash script for calculating simple interest.

## Formula
**Simple Interest = (Principal × Rate × Time) / 100**

## Features
- Interactive command-line interface
- User input for principal amount, interest rate, and time period
- Input validation
- Precise floating-point calculations

## Usage
\`\`\`bash
./simple-interest.sh
\`\`\`

## Example Calculation
\`\`\`
Enter principal amount: 1000
Enter annual interest rate (%): 5
Enter time period (years): 2

Simple Interest: $100.00
Total Amount: $1100.00
\`\`\`

## Script Information
- **File:** `simple-interest.sh`
- **Language:** Bash
- **Dependencies:** `bc` calculator
